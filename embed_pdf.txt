To embed pdfs, get the google doc embed link from http://itools.com/tool/google-docs-viewer-pdf-ppt. Input would be for instance - http://egsc.seas.columbia.edu/documents/constitution.pdf.

Example embed:

<object data="documents/constitution.pdf" type="application/pdf" width="100%" height="1000px">
	<iframe src="https://docs.google.com/viewer?url=http%3A%2F%2Fegsc.seas.columbia.edu%2Fdocuments%2Fconstitution.pdf&embedded=true" width="100%" height="1000px">
		This browser does not support PDFs. Please download the PDF to view it: <a href="documents/constitution.pdf">Download PDF</a>.
	</iframe>
</object>
